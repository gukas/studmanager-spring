На главную

Простые запросы

1. Выбрать имя, фамилию, оценку для каждого студента.

2. Выбрать имя, фамилию, оценку, номер группы для каждого студента. Сортировка по фамилии.

3. Выбрать имя, фамилию, дату рождения, оценку, номер группы, номер и название кафедры для каждого студента. Сортировка по фамилии и дате рождения (2мя способами).

4. Выбрать имя, фамилию и номер группы иностранных студентов.

5. Посчитать количество студентов.

6. Посчитать количество всех студентов, у которых оценка 4.

7. Посчитать количество студенток, у которых оценка 4.

8. Выбрать оценки и количество студентов, получивших данные оценки.

9. Вывести название кафедр и количество специальностей для каждой кафедры.

10. Вывести среднюю оценку в каждой группе.

11. Посчитать родственников для каждого студента.

12. Вычислить возраст для каждого студента. Вывести фамилию и имя студента в формате «Иванов И.» и соответствующий ему возраст.

13. Посчитать количество всех студентов старше 22 лет, у которых оценка 3.

14. Выбрать студентов, у которых родственники не выезжали за границу.

15. Посчитать студентов в каждой группе, у которых родственники не выезжали за границу.

2. С помощью запроса выбрать пропущенные номера из таблицы NUMTABLE.
